# Changelog

All notable changes to this project will be documented in this file.

This project follows [Semantic Versioning](https://semver.org/).

## [v0.1.0] - 2025-11-17
Initial public alpha release.

### Added
- `runpilot run` command for executing training jobs
- `runpilot list` for viewing run history
- `runpilot show` for inspecting run metadata
- Filesystem based run storage under `~/.runpilot/runs`
- `run.json` metadata format
- `logs.txt` logging per run
- Docker-based execution with fallback
- Early metrics extraction architecture
- Unit test coverage across core modules
- Initial documentation and project roadmap
